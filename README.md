🛠️ Configuration
Required Variables
Create a .env file or set these environment variables:

env

API_TOKEN=your_bot_token_from_botfather
ADMIN_ID=your_telegram_user_id
FORCE_JOIN_CHANNEL=your_channel_username
Variable	Description	Example
API_TOKEN	Your bot token from @BotFather	123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
ADMIN_ID	Your Telegram user ID	123456789
FORCE_JOIN_CHANNEL	Channel username (without @)	mychannel
